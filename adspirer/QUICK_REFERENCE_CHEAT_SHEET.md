## 📊 Success Metrics by Phase

### Phase 1: Learning (Days 1-3)
| Metric | Target | Red Flag |
|--------|--------|----------|
| **Impressions** | 500-2K | <100 (expand audience) |
| **CTR** | >0.5% | <0.5% (ad relevance issue) |
| **Conversions** | 1-3 | 0 in 48h (check tracking) |
| **Quality Score** (Google) | 7+ | <5 (keyword/copy mismatch) |

### Phase 2: Stabilization (Days 3-14)
| Metric | Target | Red Flag |
|--------|--------|----------|
| **CTR** | 1-2% | <0.8% (pause low performers) |
| **CPA** | Stabilizing | >3x target (targeting issue) |
| **Conversions** | 10+ | <5 (keep running, more data needed) |
| **Frequency** (Meta) | <2 | >3 (creative fatigue, rotate) |

### Phase 3: Optimization (Week 2-4)
| Metric | Target | Red Flag |
|--------|--------|----------|
| **ROAS** | >1.5x | <1.0x (pause campaign) |
| **CPA** | -30% vs Phase 1 | Trending up (expand audience) |
| **CTR** | 2%+ | <1.2% (refresh creative) |
| **Conversion Rate** | 2-5% | <1% (landing page issue) |

### Phase 4: Scaling (Week 4+)
| Metric | Target | Red Flag |
|--------|--------|----------|
| **ROAS** | Sustained >2x | Dropping (fatigue or saturation) |
| **CPA** | Stable or improving | Rising >10% week-over-week (pause) |
| **Budget efficiency** | Scaling 20-30% | Can't scale without CPA rising (budget cap) |

## 🎯 Red Flags & Quick Fixes

| Issue | Cause | Fix |
|-------|-------|-----|
| **0 conversions after 48h** | Tracking broken OR landing page issue | ✅ Verify pixel fires; don't pause |
| **CTR <0.5%** | Ad irrelevant or audience wrong | ✅ Test new headlines; narrow audience |
| **CPA >5x target** | Landing page friction OR audience too broad | ✅ Review landing page; tighten targeting |
| **No impressions after 24h** | Audience too narrow OR bid too low | ✅ Expand location/age; increase bid |
| **Frequency >3 (Meta)** | Creative saturation | ✅ Rotate to new ad variations |
| **CTR drops after week 1** | Same people seeing same ad | ✅ Launch new creative variations |

**Before I create ANY campaign, you must answer:**

- [ ] **Conversion goal?** (purchase, form, lead, signup, phone call)
- [ ] **Daily budget?** (e.g., $15, $50, $100)
- [ ] **Landing page ready?** (mobile responsive, fast load, clear CTA)
- [ ] **Conversion tracking installed?** (pixel firing on thank you page)
- [ ] **Platform choice?** (Google / Meta / LinkedIn / TikTok)
- [ ] **Creative assets ready?** (if Meta/LinkedIn/TikTok, else Google uses text only)
- [ ] **Target audience?** (location, age, interests, job title)
- [ ] **Expected ROAS or CPA target?** (what's success to you)

**If any box is unchecked → I'll ask before creating**

## 🔄 Campaign Phases at a Glance

```
LAUNCH (Day 0)
    ↓
    Campaign created PAUSED
    You review & approve
    Campaign goes ENABLED
    ↓
LEARNING PHASE (Days 1-3)
    Monitor: impressions, CTR, early conversions
    ✓ Verify conversion tracking works
    ✗ Don't pause underperformers yet
    ↓
STABILIZATION (Days 3-14)
    Add negative keywords (stop junk traffic)
    Pause ads <1% CTR
    Test headline/description variations
    ↓
OPTIMIZATION (Week 2-4)
    Pause keywords: low CTR + high CPC
    Scale winners: +10-15% bid
    Analyze ROAS, suggest budget changes
    ↓
SCALING (Week 4+)
    Budget +20-30% if ROAS >2x
    Pause losers (ROAS <1x)
    Add new keywords/audiences based on winners
```

## 🚨 Emergency Escalation

| Issue | Action |
|-------|--------|
| **0 conversions after week 1** | Pause, analyze landing page & tracking |
| **CPA >5x target for 3 days** | Pause, narrow targeting |
| **Budget unexpectedly high** | Pause, review bids & targeting |
| **Conversion tracking down** | Pause immediately, notify you |

**Rule:** When in doubt, I pause and ask. Better safe than sorry with real money.

## 💡 Pro Tips

| Tip | Why |
|-----|-----|
| **Start with BROAD keywords, convert high-performers to EXACT** | Learn fast, then optimize |
| **Always test 3-5 ad variations** | Find winners, retire losers |
| **Wait 50+ conversions before switching bid strategy** | Platform needs data to optimize |
| **Budget +20-30% every 3 days if ROAS >2x** | Scalable growth window |
| **Monitor landing page bounce rate** | CTR high but conversions low = landing page issue |
| **Never pause a campaign during learning phase** | AI still collecting data |
| **Add negative keywords aggressively** | Stop wasting money on junk |

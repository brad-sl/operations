# Stop-Slop Integration Guide

**Purpose:** Remove AI-generated prose patterns from all external marketing content (ad copy, headlines, blog posts, client recommendations).

**Status:** ✅ Integrated into Creative Agent workflow (Creative Optimizer runs Fridays at 6 PM PT)

---

## What's Deployed

### 1. **stop-slop-rules.md**
Core anti-pattern library:
- 30+ banned phrases (clichés, jargon)
- 8 structural anti-patterns (binary contrasts, passive voice, etc.)
- 5-dimension quality rating system
- Character limits for each ad platform

### 2. **StopSlopFilter.ts**
TypeScript implementation:
- `evaluate(text)` → returns quality score + issues + suggestions
- `formatReport(text)` → readable output for humans
- Checks:
  - Banned phrase matching
  - Forbidden punctuation/patterns
  - Passive voice detection
  - Binary contrast detection
  - Sentence rhythm analysis
  - Directness rating
  - Trust rating
  - Authenticity rating
  - Density rating

### 3. **Integration Point: Creative Optimizer**
- Runs: **Fridays at 6:00 PM PT** (via cron in admin_schedules.json)
- Validates all ad copy (headlines, descriptions, callouts, primary_text)
- Minimum quality score: 40/50
- Flags copy below threshold + suggests fixes
- Reports to Telegram alerts

---

## How It Works

### Flow: Campaign Creation → Stop-Slop Filter → Upload

```
CopyBot generates 30 headlines + 30 descriptions
     ↓
Stop-Slop Filter evaluates each one
     ↓
Quality < 40? → Flag + suggest revision
Quality ≥ 40? → Approve for upload
     ↓
Validated copy → Google Ads / Meta Ads via Adspirer
```

### Quality Scoring

**Total Score = (Directness + Rhythm + Trust + Authenticity + Density) / 5**

| Dimension | Scale | What It Measures |
|-----------|-------|-----------------|
| **Directness** | 1-10 | Statements vs questions/suggestions |
| **Rhythm** | 1-10 | Sentence length variation (not metronomic) |
| **Trust** | 1-10 | Respects reader intelligence (no jargon) |
| **Authenticity** | 1-10 | Sounds human, not corporate |
| **Density** | 1-10 | No filler words or bloat |

**Pass threshold:** 40/50 (average 8/10 across dimensions)  
**Fail threshold:** Below 40 → Revise before upload

## Example Usage

### In Your Creative Agent

```typescript
import StopSlopFilter from './StopSlopFilter.ts';

const filter = new StopSlopFilter();

// Headlines generated by CopyBot
const headlines = [
  "Unlock Your Potential Today",
  "Boost Sales 40% This Quarter",
  "Real-time Analytics Built for You"
];

headlines.forEach(headline => {
  const score = filter.evaluate(headline);
  
  if (!score.passed) {
    console.log(filter.formatReport(headline, 50));
  } else {
    console.log(`✅ PASS: "${headline}" (${score.total}/50)`);
  }
});
```

**Output:**

```
✅ PASS: "Boost Sales 40% This Quarter" (43/50)

📋 STOP-SLOP EVALUATION
Text: "Unlock Your Potential Today"

📊 SCORES (1-10):
  Directness:    6/10
  Rhythm:        8/10
  Trust:         7/10
  Authenticity:  5/10 (← "Unlock" is cliché)
  Density:       9/10
  ───────────────────
  TOTAL:         35/50 ❌ FAIL

⚠️  ISSUES:
  • Remove cliché: "unlock"

💡 SUGGESTIONS:
  • Use conversational, human language

→ Revise to: "Open Your Potential Today" or "Build Better Results"
```

Stop-Slop respects character limits while maintaining quality:

| Platform | Field | Max Chars | Quality Check |
|----------|-------|-----------|---------------|
| **Google Ads** | Headline | 30 | Directness + Trust |
| | Description | 90 | All 5 dimensions |
| | Callout | 25 | Density + Directness |
| **Meta Ads** | Primary text | 125 | Authenticity + Density |
| | Headline | 40 | Directness |

```typescript
// After CopyBot generates headlines, before upload:
const filter = new StopSlopFilter();
const validatedHeadlines = rawHeadlines.filter(h => {
  const score = filter.evaluate(h);
  return score.passed; // Only upload if quality ≥ 40
});

if (validatedHeadlines.length < rawHeadlines.length) {
  console.warn(`Rejected ${rawHeadlines.length - validatedHeadlines.length} low-quality headlines`);
}

// Upload only validated copy to Google Ads
await uploadHeadlines(validatedHeadlines);
```

## Quality Audit Checklist

Use this before finalizing any external content:

- [ ] No banned phrases (check stop-slop-rules.md)
- [ ] No rhetorical questions ("What if..." "Have you ever...")
- [ ] No passive voice ("is optimized by..." → "we optimize...")
- [ ] No binary contrasts ("not X, but Y")
- [ ] Active voice throughout
- [ ] Varied sentence length (not metronomic)
- [ ] No corporate jargon (synergy, ecosystem, paradigm)
- [ ] No filler words (very, really, actually, just)
- [ ] Direct statements (not suggestions/maybes)
- [ ] Sounds human, not AI
